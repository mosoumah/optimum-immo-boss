# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please follow the instructions below:

1. **Do not disclose** the vulnerability publicly until the issue has been resolved.
2. **Email** us a description of the vulnerability at [security@optimum-immo-boss.com](mailto:security@optimum-immo-boss.com).
3. Include relevant details such as:
   - A description of the vulnerability
   - Steps to reproduce
   - Severity level
   - Any other relevant information that can help us understand the issue.

## Security Updates
We will make every effort to address reported vulnerabilities as soon as possible. To stay updated on security fixes and updates, please sign up for our mailing list or check our repository regularly.
